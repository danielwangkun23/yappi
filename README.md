# ⭐ Yappi

> *Every day, a little brighter!* 每天都更闪亮！

Yappi 是一款专为 3-12 岁孩子设计的正向激励 H5 应用。通过完成每日好习惯任务获得星星，兑换奖励，解锁成就勋章，配合 AI 智能夸夸功能，让成长充满乐趣！

## 功能特色

- ✅ 每日任务打卡（刷牙、阅读、整理、运动等）
- ⭐ 星星积分系统，实时同步云端
- 🛍️ 奖励商店，用星星兑换心愿
- 🏆 成就勋章系统
- 🤖 AI 智能夸夸（Claude 驱动）
- 📊 家长数据管理面板
- ☁️ Supabase 云端存储，多设备同步

## 技术栈

- 纯 HTML / CSS / JavaScript（无框架）
- [Supabase](https://supabase.com) — 云数据库
- [Anthropic Claude](https://anthropic.com) — AI 夸夸功能
- [Vercel](https://vercel.com) — 部署托管

## 部署步骤

1. 在 Supabase 执行 `supabase_setup.sql` 建表
2. 在 `index.html` 中填入你的 Anthropic API Key
3. 推送到 GitHub，在 Vercel 一键 Import 部署
